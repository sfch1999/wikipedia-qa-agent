# AI transcripts

Full Claude Code interaction history from development. API keys are redacted
(`sk-ant-REDACTED`).

- **[`build-session.md`](build-session.md)** — the complete development
  session as readable Markdown (renders directly on GitHub). Tool inputs and
  results are truncated for readability.
- **`build-session/`** — the same session rendered untruncated to HTML with
  [claude-code-transcripts](https://github.com/simonw/claude-code-transcripts);
  clone and open `index.html` for the full-fidelity version.
